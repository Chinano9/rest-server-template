    (route) UserLogin

    Route:
    MethodPath
    POST api/auth/login

    Crea un login valido usando autenticacion propia

    Body Parameters:
    Name Type Description
    correo string Correo registrado en la base de datos en un usuario activo
    password string Contraseña del usuario que desea registrarse.
    Response:
    Name Type Description
    json Object Objeto que contiene toda la respuesta
    msg string Mensaje de respuesta para mayor entendimiento
    usuario object Informacion del usuario
    token string JSON Web Token valido para 4 horas
    Response Code:
    Type Description
    200 Si el usuario esta autenticado correctamente
    400 Si el usuario o contraseña no son correctos
    403 Si el usuario no esta activo en la base de datos
    500 Error interno del servidor
