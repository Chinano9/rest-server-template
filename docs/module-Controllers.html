    (async, inner) googleSignIn(req, res) → {void}

    Funcion que realiza el inicio de sesion de un usuario con Google SignIn. Si el usuario existe comprueba si este esta bloqueado, de ser así le niega el acceso, de no estarlo lo logea. Si el usuario no existe lo crea, usando la imagen, el correo y el nombre proporcionados, ademas de asignarle el rol USER_ROLpor defecto.

    (async, inner) login(req, res) → {void}

    Funcion que inicia sesion comprobando las credenciales proporcionadas en el body. Si el correo no existe, el usuario esta bloqueado o la contraseña es incorrecta no permite el acceso. Si las credenciales son correctas admite al usuario y genera un JWT.
